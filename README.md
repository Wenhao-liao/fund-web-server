# 基金实时估值工具

一个基于 Node.js + Express 的基金实时估值 Web 应用，支持 PWA，可添加到手机桌面使用。

## 功能介绍

### 基金管理
- 输入 6 位基金代码添加基金，自动校验有效性
- 删除已添加的基金
- 基金列表持久化存储（localStorage），刷新页面不丢失

### 实时数据展示
- 每张卡片显示：基金名称、当前估值、涨跌幅（红涨绿跌）、更新时间
- 每 60 秒自动刷新所有基金数据

### 迷你走势图
- 每张卡片内嵌小折线图，展示近 1 小时内估值走势
- 每分钟记录一次估值，最多保留 60 个数据点
- 走势数据存入 localStorage，跨刷新保留

### 历史净值图
- 点击任意基金卡片，页面底部渲染该基金的完整历史净值折线图

### 排序
- 点击"按涨跌排序"按钮，在 无序 → 涨幅降序 → 涨幅升序 三种模式间切换

### PWA 支持
- 包含 `manifest.json` 和 `sw.js`，支持添加到手机桌面作为 App 使用

---

## 技术栈

- **后端**：Node.js + Express
- **前端**：原生 HTML / CSS / JavaScript
- **图表**：Chart.js + chartjs-adapter-date-fns
- **数据来源**：天天基金（实时估值）、东方财富（历史净值）

## 接口说明

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/fund/:code` | 获取指定基金实时估值 |
| GET | `/api/history/:code` | 获取指定基金历史净值 |

## 快速启动

```bash
# 安装依赖
npm install

# 启动服务（默认端口 3000）
npm start
```

浏览器访问 `http://localhost:3000`
